# 🚀 简易构建说明

由于网络问题无法自动下载 JDK，请按以下步骤手动操作：

## 📥 步骤1：下载 JDK 17

### 方法A：官网下载（推荐）

访问：**https://adoptium.net/zh-CN/temurin/releases/**

1. 在页面中找到 **Temurin 17 (LTS)**
2. 点击右侧的 **下载** 按钮
3. 或者直接点击这个链接：
   ```
   https://github.com/adoptium/temurin17-binaries/releases
   ```
4. 找到最新版本（如 `17.0.12+7`）
5. 下载文件：`OpenJDK17U-jdk_x64_windows_hotspot_17.0.X_X.zip`

### 方法B：备用下载源

**腾讯云镜像：**
```
https://mirrors.cloud.tencent.com/AdoptOpenJDK/17/jdk/x64/windows/
```

**网易镜像：**
```
http://mirrors.163.com/java/jdk/17/
```

## 📂 步骤2：放置 JDK

下载完成后，有两种方式：

### 方式1：让脚本自动解压
```
1. 将下载的 zip 文件重命名为：jdk17.zip
2. 放到：C:\Desktop\recoder\Simple-Voice-Recorder\tools\
3. 双击运行：build_with_jdk.bat
```

### 方式2：手动解压
```
1. 解压下载的 zip 文件
2. 将解压出的文件夹（如：jdk-17.0.12+7）
3. 移动到：C:\Desktop\recoder\Simple-Voice-Recorder\tools\
4. 双击运行：build_with_jdk.bat
```

## 🔨 步骤3：构建 APK

运行：
```
build_with_jdk.bat
```

脚本会自动检测 JDK 并开始构建。

## ❓ 遇到问题？

### Q: 下载太慢怎么办？
A: 
- 使用代理/VPN
- 或者使用百度网盘等国内分享
- 或者找朋友帮忙下载后传给你

### Q: 没有 tools 文件夹？
A: 
手动创建：
```
mkdir tools
```

### Q: 不知道 JDK 放对了没有？
A: 
打开 PowerShell，检查路径：
```powershell
dir C:\Desktop\recoder\Simple-Voice-Recorder\tools\
```

应该能看到 `jdk-17.x.x` 文件夹或 `jdk17.zip` 文件
